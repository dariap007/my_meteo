# Roadmap По Чанкам

## Chunk 1. Каркас проекта

Готово:

- Gradle Android project;
- Java package `ru.student.mymeteo`;
- базовые цвета;
- Manifest и разрешения.

## Chunk 2. UI MVP

Готово:

- экран "Сегодня";
- экран "Дневник";
- экран "Профиль";
- экран "Советы";
- нижняя навигация.

## Chunk 3. Данные и risk-engine

Готово:

- Open-Meteo API;
- NOAA SWPC API;
- расчет индекса риска;
- объяснения причин риска.

## Chunk 4. Локальное хранение

Готово:

- профиль в SQLite;
- дневник в SQLite;
- таблицы `profile` и `diary_entries` через `SQLiteOpenHelper`.

## Chunk 5. Уведомления

Готово:

- канал уведомлений;
- предупреждение при среднем или высоком риске;
- тестовое уведомление на экране советов.

Следующий шаг:

- добавить WorkManager или AlarmManager для фоновой проверки прогноза.

## Chunk 6. Полировка перед защитой

Сделать:

- выбор города;
- скриншоты экранов;
- тест-кейсы;
- список источников рекомендаций;
- диаграмму архитектуры в отчете.
