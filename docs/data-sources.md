# Бесплатные источники данных

## Open-Meteo

Используется для прогноза погоды:

```text
https://api.open-meteo.com/v1/forecast
```

Запрашиваемые параметры:

- `temperature_2m`;
- `relative_humidity_2m`;
- `pressure_msl`.

Причина выбора:

- не нужен API key;
- подходит для учебного проекта;
- возвращает JSON;
- есть почасовой прогноз.

## NOAA SWPC

Используется для геомагнитной активности:

```text
https://services.swpc.noaa.gov/products/noaa-planetary-k-index-forecast.json
```

Причина выбора:

- открытые данные;
- не нужен API key;
- можно получить Kp index для оценки магнитных бурь.

## Медицинские рекомендации

В приложении используются осторожные общие советы:

- снизить нагрузку;
- соблюдать питьевой режим;
- следить за самочувствием;
- лекарства принимать только по назначению врача.

Для защиты проекта стоит добавить в отчет официальный или учебный медицинский справочник, из которого взяты формулировки рекомендаций.
