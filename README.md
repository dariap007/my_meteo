# MyMeteo

MyMeteo - студенческое Android-приложение на Java для людей с метеозависимостью. Приложение показывает индекс метеориска, объясняет причины риска, хранит дневник самочувствия и дает осторожные рекомендации.

## Что уже реализовано

- Android-приложение на Java.
- Современный спокойный интерфейс без платных и тяжелых зависимостей.
- Основные цвета: `#212842` и `#F0E7D5`.
- Бесплатный прогноз погоды через Open-Meteo.
- Бесплатный прогноз Kp index через NOAA SWPC.
- Risk-engine для расчета индекса 0-100.
- Учет профиля пользователя.
- Дневник самочувствия с быстрым выбором симптомов.
- Автоматическая привязка погоды, давления в мм рт. ст. и Kp-index к дневниковой записи.
- Локальная SQLite-база без Room и без внешних библиотек.
- Настройки уведомлений по триггерам: давление, температура, влажность, магнитные бури.
- Локальные уведомления о среднем и высоком риске.
- Дисклеймер о том, что приложение не заменяет врача.
- График динамики индекса на сегодня и завтра.
- Demo fallback, если сеть недоступна.

## Почему без платных сервисов

Проект рассчитан на учебную работу без финансирования:

- Open-Meteo не требует API key для базового некоммерческого использования.
- NOAA SWPC публикует геомагнитные JSON-данные без API key.
- Нет платных карт, авторизации, облачной базы и коммерческих SDK.
- Нет обязательного backend: MVP можно демонстрировать прямо с телефона или эмулятора.

## Запуск

1. Откройте папку проекта в Android Studio.
2. Дождитесь синхронизации Gradle.
3. Запустите конфигурацию `app` на эмуляторе или Android-устройстве.

Если Android Studio попросит установить SDK Platform, установите бесплатный Android SDK через стандартный SDK Manager.

## Архитектура

```text
MainActivity
  -> data
     -> WeatherApiClient
     -> SpaceWeatherApiClient
     -> ProfileStore
     -> DiaryStore
  -> domain
     -> RiskEngine
     -> WeatherSnapshot
     -> GeomagneticSnapshot
     -> RiskForecast
     -> UserProfile
  -> notifications
     -> MyMeteoNotifier
```

Для учебного MVP интерфейс собран программно в `MainActivity`, чтобы не подключать сторонние UI-библиотеки. При расширении проекта можно вынести экраны в отдельные Activity/Fragment или перейти на XML-разметку.

## Основные экраны

- Сегодня: индекс риска, погода, давление, влажность, Kp index, причины риска.
- Дневник: добавление записей о самочувствии, симптомах и заметках.
- Профиль: персональные факторы риска.
- Советы: рекомендации на основе текущего прогноза.
